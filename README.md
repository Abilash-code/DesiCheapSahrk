This is my project DesiCheapShark , it is my attempt to replicate the function of CheapShark a website that lists prices for games across marketplaces.
the original CheapShark offers an API but only works in USD and other select currencies, DesiCheapShark aim's to fill in this market gap.
The project uses BeautifullSoup and Playwright, beautifulsoup is a webscraping library and playwright is a browser automation tool, used for testing but 
can also be used for scraping. the IGDB API is also used to get the necessary links.
